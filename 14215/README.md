백준 {14215} - {세 막대}



문제 링크: https://www.acmicpc.net/problem/14215



풀이 날짜: 2025-09-13



사용 언어: C



✏️ 풀이 아이디어



세 변을 오름차순으로 정렬해 x ≤ y ≤ z라 하자.

삼각형 조건은 x + y > z.



이미 x + y > z면: 그대로 둘레 x + y + z가 최대.



아니라면(x + y ≤ z): 가장 긴 변 z를 \*\*x + y - 1\*\*로 줄이면 넓이가 양수인 삼각형이 되며, 이때 둘레가 최대가 됨.

둘레 = x + y + (x + y - 1) = 2\*(x + y) - 1.
